Newman CLI

Newman allows us to run Postman collections from the command line and generate different types of test reports, such as CLI, JSON, HTML, or JUnit.

It also enables seamless integration of Postman API tests into CI/CD pipelines like Jenkins, GitLab CI, Azure DevOps, etc., making it suitable for automated testing workflows.
