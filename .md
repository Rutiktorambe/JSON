# 1. Quick Introduction to Python

## Definition

**Python** is a high-level, interpreted programming language known for its **simplicity, readability, and versatility**.

It is widely used in:

- Web Development
- Data Science
- Automation
- Artificial Intelligence
- Machine Learning
- Scripting
- DevOps tools

Python was created by **Guido van Rossum** and released in **1991**.

The philosophy of Python focuses on **writing simple and readable code**.

---

## Why Companies Use Python

Python is popular in offices and tech companies because:

- Easy to learn
- Clean syntax
- Large community
- Huge library ecosystem
- Works for automation and large systems

Many large companies use Python including:

- Google
- Netflix
- Instagram
- Spotify
- Dropbox

# 2. Variables

## Definition

A **variable** is a named container used to store data so that it can be reused or modified later in a program.

Think of variables like **labeled folders in an office cabinet**.

Example

```python
employee_name = "Rahul"
salary = 50000
```

---

## Key Notes

- Variables store values
- Python does **not require type declaration**
- Variable names should be meaningful
- Python is **case sensitive**

Example:

```
salary ≠ Salary
```

---

## Variable Naming Rules

Allowed

```
employee_name
salary
project_deadline
```

Not allowed

```
1employee
salary$
class
```

---

# 3. Data Types and Operators

## Definition

**Data Types** define the type of value stored in a variable.

Python automatically determines the data type.

---

## Common Data Types

| Data Type | Example         | Description         |
| --------- | --------------- | ------------------- |
| int       | 10              | Whole number        |
| float     | 10.5            | Decimal number      |
| str       | "Hello"         | Text                |
| bool      | True            | Boolean value       |
| list      | [1,2,3]         | Collection of items |
| dict      | {"name":"John"} | Key value pairs     |

---

## Example

```python
employee = "Rahul"
salary = 50000
bonus = 5000.50
is_manager = False
```

---

# Operators

Operators perform operations on variables.

---

## Arithmetic Operators

| Operator | Meaning        |
| -------- | -------------- |
| +        | Addition       |
| -        | Subtraction    |
| \*       | Multiplication |
| /        | Division       |
| %        | Modulus        |
| \*\*     | Power          |

Example

```python
salary = 50000
bonus = 5000

total = salary + bonus

print(total)
```

Output

```
55000
```

---

## Comparison Operators

| Operator | Meaning          |
| -------- | ---------------- |
| ==       | Equal            |
| !=       | Not equal        |
| >        | Greater          |
| <        | Less             |
| >=       | Greater or equal |
| <=       | Less or equal    |

Example

```python
tasks_done = 6
required_tasks = 5

print(tasks_done > required_tasks)
```

Output

```
True
```

---

## Logical Operators

| Operator | Meaning              |
| -------- | -------------------- |
| and      | Both conditions true |
| or       | One condition true   |
| not      | Reverse result       |

Example

```python
has_laptop = True
has_wifi = True

print(has_laptop and has_wifi)
```

# 4. Functions

## Definition

A **function** is a reusable block of code used to perform a specific task.

Functions help:

- Avoid repeating code
- Improve readability
- Organize programs

---

## Syntax

```python
def function_name():
    code
```

---

## Example

```python
def greet():
    print("Welcome to the company!")
```

Calling the function:

```python
greet()
```

---

## Function With Parameters

```python
def greet_employee(name):
    print("Welcome", name)
```

Call

```python
greet_employee("Rahul")
```

---

## Function Returning Value

```python
def calculate_salary(base, bonus):
    total = base + bonus
    return total
```

Use it

```python
salary = calculate_salary(50000, 7000)

print(salary)
```

---

## Funny Office Example

Employee motivation calculator.

```python
def motivation_level(meetings):
    motivation = 100 - meetings * 10
    return motivation

print(motivation_level(4))
```

Meaning: More meetings = Less motivation.

# 5. Dictionaries and Dictionary Manipulation

## Definition

A **dictionary** stores data in **key-value pairs**.

Example

```
key : value
```

Used for structured information like employee data.

---

## Example

```python
employee = {
    "name": "Rahul",
    "department": "IT",
    "salary": 60000
}
```

---

## Accessing Values

```python
print(employee["name"])
```

---

## Adding Data

```python
employee["age"] = 28
```

---

## Updating Data

```python
employee["salary"] = 65000
```

---

## Removing Data

```python
employee.pop("age")
```

---

## Looping Through Dictionary

```python
employee = {
    "name": "Rahul",
    "department": "IT",
    "salary": 60000
}

for key, value in employee.items():
    print(key, value)
```
